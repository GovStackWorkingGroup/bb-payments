# 8.5 Business to Government (B2G) payments

For Business to Government (B2G) transactions the following existing APIs would need to be used:

For Invoice Payments

* [billInquiryBiller](8.3-person-to-government-apis-p2g-bill-payments.md#id-8.3.1-billinquirybiller) - The existing API to retrieve invoice details detailed in the [P2G payments](8.3-person-to-government-apis-p2g-bill-payments.md) would be used to get bill details from the Bill Aggregator or Govt Entity.
* [markBillPayment](8.3-person-to-government-apis-p2g-bill-payments.md#id-8.3.3-markbillpayment) - The existing API to bill payment detailed in the [P2G payments](8.3-person-to-government-apis-p2g-bill-payments.md) would be used to mark the invoice as paid.

For RTP Payments

* [billerRtpReq](8.3-person-to-government-apis-p2g-bill-payments.md#id-8.3.4-billerrtpreq) - The existing API for to initiate a "Request to Pay" (RTP) as detailed in the [P2G Payments](8.3-person-to-government-apis-p2g-bill-payments.md) would be used try the Bill Aggregator or Govt Entity to initiate the RTP.
* [markBillPayment](8.3-person-to-government-apis-p2g-bill-payments.md#id-8.3.3-markbillpayment) - The existing API to bill payment detailed in the [P2G payments](8.3-person-to-government-apis-p2g-bill-payments.md) would be used to mark the invoice as paid.
