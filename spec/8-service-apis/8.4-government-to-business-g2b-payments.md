# 8.4 Government to Business (G2B) payments

For Government to Business (G2B) transactions the following existing APIs would need to be used:

* Register Beneficiary - The existing Register Beneficiary API detailed in [G2P payments ](8.1-government-to-person-g2p-payments.md)would be able to be used to register a business payment details in the ID Account Mapper, assuming a functional ID for a business can be allocated by the ID BB.
* Update Beneficiary - The existing Update Beneficiary API detailed in [G2P payments](8.1-government-to-person-g2p-payments.md) would be able to be used to update a registered Business payment details in the ID Account Mapper.
* Bulk Disbursement - The existing Bulk disbursement APIs detailed in [G2P payments](8.1-government-to-person-g2p-payments.md) would be able to be used to send a batch of credit instructions for businesses
